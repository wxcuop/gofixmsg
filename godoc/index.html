GoFixMsg

FIX (Financial Information eXchange) Protocol Implementation in Go

GitHub Repository pkg.go.dev Main Docs

📚 Overview

GoFixMsg is a comprehensive FIX session management library for Go, providing:

  • Message Parsing & Creation: Parse FIX messages from wire format, manipulate fields, serialize back
  • Session Management: Full FIX session lifecycle with state machine
  • Persistence: SQLite-based message store for gap filling and recovery
  • Multi-Session: Concurrent session handling with initiator and acceptor modes
  • Spec Support: Load QuickFIX XML specifications for repeating group support

📦 Core Packages

engine

FIX session engine and state management. Core coordinator for session lifecycle.

fixmsg

FIX message parsing, creation, and field access. In-memory message representation.

fixmsg/codec

Wire format encoding/decoding. Converts between bytes and FixMessage objects.

fixmsg/spec

QuickFIX XML specification loading for repeating group and message type support.

network

TCP/TLS connection management. Initiator and Acceptor modes for FIX connections.

state

Session state machine. Manages FIX session lifecycle transitions.

store

Message persistence layer using SQLite. Enables gap filling and recovery.

handler

Message handler framework for custom FIX processing.

heartbeat

Heartbeat and test request monitoring and management.
